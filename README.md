# asgardeo-agent-auth-ts

 - This includes typeScript examples on how to secure AI agents using Asgardeo
 - Examples contain 2 main authencation flows,
   1. [AI agent acting on its own](https://wso2.com/asgardeo/docs/guides/agentic-ai/ai-agents/agent-authentication/#ai-agent-acting-on-its-own)
   2. [AI agent acting on behalf of a user](https://wso2.com/asgardeo/docs/guides/agentic-ai/ai-agents/agent-authentication/#ai-agent-acting-on-behalf-of-a-user)
 - Also, there are examples for following AI agent development frameworks,
   1. LangChain
   2. Google ADK
